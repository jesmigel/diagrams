## Make commands
| Command | Description |
| - | - |
| build | Builds up the docker image |
| up | Spins up a container |
| down | Stops and removes the containers as well as volumes |
| logs | Prints the container logs |
| status | Prints the container states |
| login | Brings up the container shell (bash) |
| test | Prints the container states |
